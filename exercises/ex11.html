Θα φτιάξουμε μια navbar

STEP 1: remove the bullets and the margins and padding from the list:

ul { list-style-type: none;
margin: 0;
padding: 0;
}

STEP 2: Ομορφύνετέ την :

ul {
list-style-type: none;
margin: 0;
padding: 0;
overflow: hidden; Μην αφήσετε αυτό το element να μεγαλώσει!!!
background-color: #333;
}

STEP 3: Αριστερή στοίχισση στα li:

li {
float: left;
}

STEP 4: Κάντε την κάθετη:

li a {
display: block; Όλη η περιοχή γίνεται clickable!
color: white;
text-align: center;
padding: 14px 16px;
text-decoration: none;
}

STEP 5: Κάποιοι όμορφοι selectors!!!:

li a:hover:not(.active) {
background-color: #111;
}
.active {
background-color: #4CAF50;
} }

STEP 6: To about στα δεξιά!:

< li style="float:right"> < a class="active" href="#about"
