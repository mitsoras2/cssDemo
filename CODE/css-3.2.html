CSS3 paddings

Αυτό το στοιχείο έχει padding attribute αλλά όχι margin στο CSS. Παρατηρήστε ότι δεν έχει κενό σε σχέση με τα γειτονικά στοιχεία, αλλά στο εσωτερικό του!
